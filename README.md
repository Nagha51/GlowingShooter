# GlowingShooter
Splash boom Zbwa !

![Glowing shooter image][glowing_shooter_image]

# Install

```bash
pip install -e .
```


# Server CLI commands

- Start server:
```bash
glowing_shooter
```

- Show help

```bash
glowing_shooter --help
```

# Access web page

```bash
npm run build
```

Open [Game Home Page: /dist/index.html](/dist/index.html)


# Development

### Python

- Install test extras
```bash
pip install -e .[test]
```

### Web

- Live-updating `game.XXX.js` and `game.XXX.css`
```bash
npm run watch
```
