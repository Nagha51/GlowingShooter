# GlowingShooter
Splash boom Zbwa !

![Glowing shooter image][glowing_shooter_image]

# Install

```bash
pip install -e .
```


# Server CLI commands

- Start server:
```bash
glowing_shooter
```

- Show help

```bash
glowing_shooter --help
```

# Access web page

Open [Game Home Page: /glowing_shooter/front/home.html](/glowing_shooter/client/home.html)


# Development

- Install test extras
```bash
pip install -e .[test]
```
